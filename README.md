# 🌌 Overview

**PredictOS** is a futuristic web-based CPU Scheduling Simulator that combines classic Operating System scheduling algorithms with AI-powered burst prediction.

Designed with a modern glassmorphism interface and interactive visualizations, PredictOS helps students and developers understand scheduling concepts through simulation, analytics, and intelligent prediction models.

---

# ✨ Features

## ⚙️ CPU Scheduling Algorithms

* 🟦 FCFS (First Come First Serve)
* 🟨 SJF (Shortest Job First)
* 🟩 Round Robin (Custom Quantum)

---

## 🤖 AI Prediction Engine

* Exponential Smoothing based prediction
* Adjustable smoothing factor (α)
* Prediction history tracking
* Accuracy monitoring

# AI-Based CPU Scheduling Simulator
### by SEVEN UP ↑

## How to Run
1. Download all 3 files: `index.html`, `style.css`, `script.js`
2. Keep them in the **same folder**
3. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
4. No server, no installation needed — fully client-side

## Features
- **FCFS, SJF, Round Robin** scheduling algorithms
- **AI Burst Prediction** using Exponential Smoothing (adaptive)
- **Interactive Gantt Chart** with hover tooltips & PNG download
- **Performance Metrics**: Waiting Time, Turnaround, Response, CPU Utilization, Throughput
- **Bar & Line Charts** for algorithm comparison
- **Dark / Light theme** toggle
- **AI Training Panel** with prediction history and accuracy
- Persistent AI memory via localStorage
